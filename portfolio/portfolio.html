React

React is an open-source JavaScript library for building user interfaces in a predictable and efficient way using declarative code. It is maintained by Facebook and a community of individual developers and companies. React can be used as a base in the development of single-page or mobile applications or to build complex apps if you utilise it with other libraries.

Ignite Technology

Ignite is a student driven, project based program where students develop professional skills through internships with Fortune 500 companies and local startups. Ignite Technology allows for students to gain real world experience on topics such as artificial intelligence, robotics, cyber security, and programming. Ignite Technology is designed to align students with future opportunities in the community and help them build meaningful relationships with mentors in business, industry, and higher education, not only to learn a profession, but also to gain valuable professional skills and entrepreneurial habits of mind necessary for careers in an innovation-based economy.
